# WEATHER WEB

This application allows you to know the current temperature, wind speed, humidity and other weather conditions of **any city** in the world.

## Table of Contents

- [Feature](#feature)

- [Usage](#usage)

- [Technologies Used](#technologies-used)

## Feature 

* Reponsive design
* Units in celsius and farenheit
* Dark and Light Mode
* Error Handling
